About cstmc

CSTMC-collection.ca is an open data application. The data obtained comes directly from the Canada Science and Technology Museums Corporation (CSTMC) Open data portal. You can find the github repository of this web application here (cstmc-django). There is also a Ruby on Rails version of this app here (cstmc_code2015).
